# Ganesha CLI Commands

| Command | Description |
|---------|-------------|
| /help | Show this help |
| /mode [code\|ask\|architect] | Switch working mode |
| /model | Show/set current model |
| /clear | Clear conversation |
| /undo | Revert last change |
| /diff | Show file diffs |
| /git <cmd> | Run git command |
| /commit [msg] | Commit staged changes |
| /add <files> | Add files to context |
| /drop <files> | Remove from context |
| /ls | List files |
| /mcp | MCP server status |
| /provider | Show/set LLM provider |
| /exit | Exit Ganesha |

**Modes:**
- **code** - Edit code directly
- **ask** - Q&A without editing  
- **architect** - Plan then edit
